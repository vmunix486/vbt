# vbt
vmunix's Battery Tray icon. For Windows XP with MSYS2/MinGW.

<img width="500" alt="batteryicon" src="https://github.com/user-attachments/assets/a41c0529-dbda-4c46-bb65-9be8fc8916b6" />


I created this because the basic power icon sucks really badly on normal Windows XP. The original only has Full, Half Full, Low, Charging, and Plugged in but not Charging, and it doesn't even have a percentage. So I changed it and here's the code for a better battery tray icon I made.

Here's some things:
| Background Color | Meaning |
| --- | --- |
| Gray | Unknown |
| Blue | Charging |
| Red | Critically low (below 10%) |
| Amber | Low (below 20%) |
| Dark Gray | Plugged in but not charging (eg. Full) |
| Green | Normally discharging, and at a healthy level |
